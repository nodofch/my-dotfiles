# 💻 Konfigurasi Lingkungan Kerja Saya (Ghostty, Nvim, Starship)

Selamat datang di repositori dotfiles yang berisi konfigurasi utama untuk lingkungan kerja berbasis terminal saya. Fokus utama dari repositori ini adalah estetika, kecepatan, dan produktivitas melalui alat-alat berikut: **Ghostty, Neovim, dan Starship**.

## 🚀 Komponen Utama

| Komponen | Deskripsi | Status |
| :--- | :--- | :--- |
| **Terminal** | [Ghostty](https://ghostty.io/) | Pengaturan tema, font, dan keybinds untuk terminal berbasis GPU yang cepat. |
| **Editor** | [Neovim (Nvim)](https://neovim.io/) | Konfigurasi LUA untuk Nvim 0.10+, menggunakan plugin manager (contoh: LazyVim/Packer) untuk IDE seperti pengalaman. |
| **Prompt** | [Starship](https://starship.rs/) | Pengaturan kustomisasi `starship.toml` untuk prompt shell yang informatif, minimalis, dan cepat. |

---

## 📂 Struktur Repositori

Repositori ini mengikuti struktur standar, dengan setiap folder mewakili satu aplikasi yang dikonfigurasi.
